The Project

About WildPack

WildPack is a passion project built during the Java cohort at Zip Code Wilmington. It started as a simple idea — an animal encyclopedia for extinct species — and grew into a full-stack Spring Boot application with a REST API, a JavaScript frontend, and a flashcard game.

The inspiration came from the news in 2025 that scientists had successfully brought back the Dire Wolf. There was no clean, modern place to read about that story alongside the rest of extinction history. So I built one.

I've always been into animals and wildlife. Transitioning from warehouse work into software development, this project gave me a chance to combine something I genuinely care about with the skills I've been building at ZCW.
